# SystemVerilog Example

This example demonstrates traceability in a simple SystemVerilog design.

## Files
- `src/design.sv`: Contains the RTL with `[REQ-xxx]` tags.
- `requirements.csv`: Defines the requirements.

## How to Run
Use the provided `run_test.bat` or `run_test.sh` script.
Output report will be generated in `../test_outputs/01_SystemVerilog_report.html`.
